This project is my starting point in understanding Embedded software engineering. For this project I worked with the AVR micro-controller ATmega128p. I used the Pocket AVR Programmer along with AVRdude to upload the program to the microcontroller. The goal of this project was to create a blinking LED light programmed by the microcontroller

Items used:

  • 1x Pocket AVR Programmer
  • 1x ATmega328P
  • 1x Breadboard
  • 1x 16 MHz crystal
  • 1x 10k resistor
  • 2x 22 picofarad capacitors
  • 1x LED

Program uploaded:

Setting all C Ports as outputs and Setting all Outputs to high and low with a second delay
